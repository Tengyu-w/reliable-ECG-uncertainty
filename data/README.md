# Dataset Access

The raw ECG dataset is not distributed in this repository.

The code expects a local file named:

```text
RHYTHMS.mat
```

The file should contain MATLAB cell arrays for the rhythm classes used in the
project:

- `SR`
- `VT`
- `VF`

The public repository only contains source code, method documentation, and
aggregate non-identifiable result summaries. Raw ECG records, window-level
signals, model checkpoints, embeddings, and private review examples are
excluded.

Before any archival release, the dataset source, licence, consent or ethics
status, and access procedure should be documented according to the data owner's
requirements.
